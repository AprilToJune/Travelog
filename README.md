# ✈️ Travelog
- Travel + Vlog 합성어 
- 우리가 여행을 갔던 곳을 기록하기 위해 만든 웹 애플리케이션

# 📌 사용된 기술 스택
- `Language`:
  - JavaScript

- `Library`: 
  - React.js
  - styled-component(CSS-in-JS)
  - Material-UI (Components)
  - kakao-map-api (Map)

- `Linter`:
  - ESlint

- `Database & Storage`:
  - Firebase firestore
  - Firebase storage

- `Deployment`:
  - Firebase 


# 📌 기능

- 사진 업로드 기능
  - 제목, 날짜, 위치, 이미지들을 입력 받고 업로드할 수 있음.
  
- 지도 마커 & 기록 개수 띄우기 기능

- 모달 기능
